How to Prepare — Before Each Module

Before M2 — Excel
Install Excel or Google Sheets. Practice SUM, IF, VLOOKUP. Your HR work already gives you a head start here.
Before M3 — SQL
Install MySQL or PostgreSQL (free). Watch the first 3–4 videos of Data with Baraa's SQL course — just SELECT is enough.
Before M4 — Stats
Watch 5–6 StatQuest videos on descriptive stats. Your HR metrics background means correlation and averages will click fast.
Before M5 — Power BI
Download Power BI Desktop (free). Connect to an Excel file and build one chart before class. It moves quickly.
Before M6 — Python
Install Python via Anaconda. Open Jupyter Notebook. Run print('hello') — that's all you need before class.
For M7 — Capstone
Plan to use an HR dataset — attrition, headcount, performance. PMUK-type data will make your portfolio stand out.
