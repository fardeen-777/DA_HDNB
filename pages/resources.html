How to use these resources

Don't try to finish them
Watch 1–2 hours before the relevant module starts — just enough to recognise concepts when they come up in class.
Code along
For SQL and Python especially, pause and type every command yourself. Watching passively is not the same as doing.
Use them for revision too
After each class, rewatch the relevant section at 1.5× speed. The second pass always lands better.
