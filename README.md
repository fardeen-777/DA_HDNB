A personal prep guide, progress tracker, and YouTube resource hub for the **HDNB Data Analytics Career Bootcamp** (June–September 2026).

Built by a participant for personal use. Not an official HDNB product.

## 🚀 How to use

**Locally:** Open `index.html` in any browser. No build step, no server needed.

**On GitHub Pages:**
1. Push to a GitHub repo
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)` folder
4. Your site will be live at `https://yourusername.github.io/repo-name/`

## ✅ Tracker features

- Dropdowns for **Class Attendance**, **Assignment**, and **Quiz / Test** per class
- Three statuses: **Done** (green) · **In Progress** (blue) · **Pending** (amber)
- Per-module progress bars + overall completion percentage
- Note-taking per class via prompt dialog
- All state auto-saved in **localStorage** — persists across browser sessions
- Reset button to clear all progress
